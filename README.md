# SC2002 Group Assignment: Turn-Based Combat Arena (Java)

##  Project Overview
This project is a command-line turn-based combat arena game developed as part of the **SC2002 (Object-Oriented Design & Programming)** course at Nanyang Technological University (NTU). 

The primary focus of this project is to demonstrate a deep understanding of **Object-Oriented Programming (OOP)** principles and the practical application of **SOLID design principles** to create an extensible, maintainable, and decoupled software architecture.

---

##  Core Features
- **Class-Based Combatants**: Play as a **Warrior** (Shield Bash/Stun) or a **Wizard** (Arcane Blast/AOE).
- **Dynamic Enemy System**: Battle against Goblins and Wolves with different attributes and speeds.
- **Advanced Logic**:
    - **Backup Spawning**: Supports multi-wave combat logic where reinforcements enter the arena after initial enemies are defeated.
    - **Status Effect System**: Handles persistent effects like Stun, Defense buffs, and Smoke Bomb invulnerability.
    - **Item System**: Single-use items (Potion, Power Stone, Smoke Bomb) to turn the tide of battle.
- **Difficulty Levels**: Three levels (Easy, Medium, Hard) with varying enemy pool specifications.

## Getting Started
### Prerequisites

Java Development Kit (JDK) 11 or higher.

### Installation & Execution

Clone the repository:
```bash
git clone https://github.com/zhangzexuanecho-droid/SC2002-Grp6-Turn-Based-Combat-Arena.git
```
Navigate to the src/app directory:
```bash
cd src/app
```
Compile the project:
```bash
javac main.java
```
Run the game:
```bash
java main
```
